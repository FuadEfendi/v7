V7



V7 is an example application demonstrating the use of Vaadin 7 for the user interface, Apache Shiro for application security, Guice for dependency injection and OrientDB for persistence.  As an application it does not do anything useful, but is just a reference for programmers using these technologies.  There are blog, forum, and documentation sites, and the source code is available under an Apache 2.0 licence.  I hope you find this useful, and your comments would be welcome on the forum or the blog.
